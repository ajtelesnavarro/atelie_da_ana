Este foi um projeto feito nas primeiras semanas de fevereiro de 2026, pelas matérias de Programação Back-End e Linguagem de Marcação.
## index.html
Tem como objetivo ser a página principal de nosso site, além de uma breve introdução ao site, abaixo pode ser visto uma explicação detalhada de como os produtos são pensados, esboçados e depois feitos.
## sobre_o_atelie.html
Escrevi uma breve introdução de como tive a ideia de criar o ateliê e um dos meus trabalhos mais reconhecidos.
## produtos.html
Pode-se vêr todos os produtos disponíveis para compra com um hyperlink direto para a parte de encomendas, logo abaixo há o preço de cada produto.
## encomendas.html
É uma página dedicada aos consumidores fazerem seus pedidos, é nos informados informações como nome completo, telefone, e-mail, senha e um hyperlink para a página de login, caso o consumidor já tenha uma conta registrada em nosso site.
Logo após o hyperlink, há uma série de opções na qual o consumidor pode escolher, entre o tipo de produto que quer, além de pedir a sua própria encomenda, e ao final dois botões para envio das informações ou remoção delas.
## feedback.html
Nesta parte, o consumidor preenche campos simples como nome e telefone, e logo abaixo há um campo para preencher com algum comentário ou indignação, com botões de limpar os dados ou envia-los
